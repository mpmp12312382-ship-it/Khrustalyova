# Random Task Generator

**Автор:** Хрусталёва Маргарита Андреевна

## Описание

приложение для генерации случайных задач. 
### Основные возможности

Приложение помогает пользователям получать случайные задания из предопределённого списка, добавлять собственные задачи и сохранять историю выполненных генераций. История сохраняется в JSON-файл и загружается при запуске.


## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.6 или выше

2. Скачайте файл `main.py`

3. Запустите приложение:
```bash
python main.py
